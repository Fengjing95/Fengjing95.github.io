# Day4

## 旋转+透明度

```css
div {
	width: 150px;
	height: 150px;
	border-radius: 50%;
	overflow: hidden;
	border: solid 3px #409eff;
	transition: 0.5s;
	opacity: 0.5;
}
div:hover {
	transform: rotate(200deg);
	opacity: 1;
}
```



今日收获。。。就是个透明`opacity`
